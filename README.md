# Calculator Project

This project is a simple console application written in Python that can perform 4 basic operations (addition, subtraction, multiplication, division).

## Usage

You must have Python 3 installed on your computer to run this project.

1.  Clone the project or download the files.
2.  Open your terminal or command prompt in the project folder.
3.  Run the following command:

```bash
python calculator.py
